# Safe Routes AI

A web application that helps students and parents in Howard County, Maryland find the safest walking and biking routes to school.

## Features

- Interactive map with real-time route planning
- Community hazard reporting (traffic, crosswalks, lighting, sidewalks)
- Howard County open data integration (schools, sidewalks, crosswalks, traffic signals)
- Safe route scoring based on hazard data
- Walk group coordination for parents

## Tech Stack

- HTML / CSS / JavaScript
- Leaflet.js (mapping)
- Valhalla (routing engine)
- Howard County Open Data API
